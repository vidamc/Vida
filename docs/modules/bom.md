# bom

Java Platform BOM (`dev.vida:vida-bom`): согласованные версии runtime-библиотек Vida для внешних мод-проектов.

- Пакет проекта: `:bom`
- Координата Maven: **`dev.vida:vida-bom:<release>`** (подставьте номер релиза Vida)
- Gradle: `java-platform` + `maven-publish`

Полное описание потребления — [platform BOM reference](../reference/platform-bom.md).

## Исходники

Логика ограничений — в корне репозитория:

`bom/build.gradle.kts`
